표(table) 관련 서식

  • class="table" : 테이블 기본서식이 적용됨.
  • class="table-striped" : table에 적용 - 각 행마다 홀,짝 색상 다르게
  • class="table-bordered" : table에 적용 - 테이블에 테두리가 생김
  • class="table-hover" : table에 적용 - 마우스 오버시 색상이 전환
  • class="table-dark" : thead에 적용 - 배경색 어둡게
  • class="text-center" : 가운데 정렬
  • class="table-success" : tr에 적용 - 색상(green)
  • class="table-danger" : tr에 적용 - 색상(red)
  • class="table-info" : tr에 적용 - 색상(bule sky)
  • class="table-primary" : tr에 적용 - 색상(bule)
  • class="table-warning" : tr에 적용 - 색상(orange)
  • class="table-responsive" : 화면 너비가 768px미만일 경우 가로로 스크롤됨

셀배경 색상 클래스는 tr, td에 각각 분류색상을 적용할 수 있음.
