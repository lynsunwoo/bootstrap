부트스트랩 - 01_이미지관련

이미지태그를 사용하여 이미지를 삽입하고 다양하게 클래스를 적용해본다.

  • class="rounded"
  • class="rounded-circle"
  • class="rounded-circle img-circle"
  • class="img-thumbnail"
  • class="img-fluid"

1. 모서리 둥글게 - css(border-radius)

부트스트랩에서는 class='rounded'으로 모서리 둥글게 적용이 가능

2. 모서리를 둥글게 하여 원이미지로 만듬.

class="rouded-circle img-circle"

3. 이미지를 갤러리처럼 썸네일 이미지로 보이게 class="img-thumbnail

class="img-thumbnail" : "border", "padding", "border-radius" 속성값 모두 적용 됨.

4. 이미지와 문자를 사용하여 콘텐츠 제작시 figure, figcaption 사용할 경우

class="img-fulid" : 이미지를 반응형으로 동작하는 클래스 부모 속성에 맞춰서 width:100%;, heigth:auto; 속성이 자동으로 들어감

부모의 크기가 변하면 이미지 크기도 같이 반응. (==반응형 이미지)
